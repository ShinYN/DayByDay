# Null-Safety

기능
- 컴파일 타임에 Null 을 체크

관련 애노테이션
- NonNull
- Nullable
- NonNullApi (Package Level)
- NonNullFields (Package Level)

Package Level 적용 방법
1. 패키지에 `package-info` 파일을 추가한다
2. 해당 파일에 애노테이션을 추가한다

Code Level 적용 방법
1. 메서드나 파라미터에 애노테이션을 추가한다
    ~~~java
    @NonNull
    public String createEvent(@NonNull String name) {
        return  "hello " + name;
    }
    ~~~
