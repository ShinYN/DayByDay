##주석
- 파이썬에서 주석은 '#'을 붙여서 작성한다. 
- 주석에 한글 표기
    - 프로그램에 한글이 들어가면 코드 앞에 '# coding=cp949' 문구가 들어가야 한다. 
        - cp949: 마이크로소프트가 정한 한글 표현 방식
    - '# coding=utf-8' 은 UTF8형식


##문자열 표시
- 작은 따옴표(') 와 큰 따옴표(")는 문자열을 지정
- 따옴표 세 개 (''')은 여러줄에 걸친 문자를 표시

##포멧팅
- '{0}'.format(value)
- ex) '{0}, {1}'.format('A', 'B')
